The following are the files required to simulate the energy center in building 4 of the housing district in Durlach, Karlsruhe.  

 
You can run them by following the following instrucitons:
* Move/paste `main_sim.py` in the `src` directory.
* Move/paste `input_params.json` in the `data/inputs` directory
* In line 191 of `main_sim.py` the name of the file containing the heatload timeseries should be changed accordingly.
